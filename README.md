import-media
====

```
Imports media files (photo/video) to a media root, organized by date.

Usage: import-media [-v|--verbose] [-n|--noop] [-i|--ignore-file-name]
                    (-m|--media-root ARG) MEDIA_FILES...

Available options:
  -h,--help                Show this help text
  -v,--verbose             be verbose
  -n,--noop                do not import files, show what would be done
  -i,--ignore-file-name    don't include original file name in the target file
  -m,--media-root ARG      media root directory
```
